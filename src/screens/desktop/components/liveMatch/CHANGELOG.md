## [Unreleased]
### Added
- Initial LiveMatch component creation combining card and table views.
- Segmented control for toggling between "Card View" and "Table View".
- Support for LiveMatchCover overlay when no match is active.
- Integration with existing MatchTabNew (card view) and MatchTable/MatchInfoCard (table view) components.
- Localization support for view mode labels.
- Styling consistent with application theme system.
