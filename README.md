Overview
Marvel Rivals Overlay is an Overwolf application that enhances your Marvel Rivals gaming experience with real-time statistics, match history tracking, and character information. The overlay provides valuable insights without disrupting your gameplay.

Features
- **In-Game Team Stats**: View your team's performance with K/D/A, final hits, damage blocked, healing, and ultimate charge information.
- **Character Swap Bar**: Track character changes during matches with an unobtrusive notification bar.
- **Final Hits Tracker**: Monitor final blows between you and enemy players.
- **Match History**: Review detailed statistics from your previous matches, including player stats, team contributions, and match outcomes.
- **Recent Players Tracking**: Keep track of recently encountered players, their characters, and encounter stats.
- **Multi-Display Support**: Works correctly across various display configurations, resolutions, and game modes.

Installation
1. Download the latest release from the Overwolf Store.
2. Install and launch the application.
3. Start Marvel Rivals and enjoy the enhanced gameplay experience.

Windows
The overlay includes several windows for different aspects of the game:

### Desktop App
The main window provides access to:
- **Current Match Information**: Displays match details such as map, game type, game mode, and player stats.
- **Match History**: A detailed view of past matches, including player performance, team stats, and match outcomes.
- **Recent Players**: Tracks recently encountered players, showing their characters, encounter stats, and last seen time.
- **Application Settings**: Customize the overlay's behavior, language, and display options.

### In-Game Overlay
Appears during active matches, showing:
- **Team Statistics**: Displays player K/D/A, final hits, damage blocked, healing, and ultimate charge.
- **Final Hits Tracking**: Tracks final blows for teamates.

### Character Swap Bar
- Displays active character swaps in real-time.

### Final Hits Bar
- Displays how many final hits the player got on each opponent and how many they got on the player.

Configuration
The overlay is designed to be unobtrusive and automatically positions itself appropriately based on your:
- Screen resolution
- Display configuration (multi-monitor setups)
- Game window mode (fullscreen or windowed)
- Rendering settings (including DLSS/upscalers)

You can customize various aspects of the overlay in the Settings menu, including:
- Language preferences (supports multiple languages)
- UI positioning
- Display options
